Build Tag Syntax Build Tag Sample Boolean Statement
Space-separated elements // +build pro enterprise pro OR enterprise
Comma-separated elements // +build pro,enterprise pro AND enterprise
Exclamation point elements // +build !pro NOT pro

下面简单说说如何构建免费版、专业版和企业版的不同功能集的程序

import "fmt"

var features = []string{
	"Free Feature #1",
	"Free Feature #2",
}

func main() {
	for _, f := range features {
		fmt.Println(">", f)
	}
}

## "pro enterprise" tag都构建
chunk@chunk-ThinkPad-X1-Carbon-5th:/tmp/go$ go build -tags "pro enterprise"
chunk@chunk-ThinkPad-X1-Carbon-5th:/tmp/go$ ./a 
> Free Feature #1
> Free Feature #2
> Enterprise Feature #1
> Enterprise Feature #2
> Pro Feature #1
> Pro Feature #2

## 只构建enterprise的tag（注意这个）
chunk@chunk-ThinkPad-X1-Carbon-5th:/tmp/go$ go build -tags enterprise
