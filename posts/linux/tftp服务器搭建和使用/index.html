快速启动tftp服务端

1
2
3

pip install py3tftp
#使用下面命令即可启动端口是69 udp的
python -m py3tftp

客户端上传文件

1
2
3
4

import tftpy

client = tftpy.TftpClient('192.168.5.xxx', 69)
client.upload('1.conf', '1.conf')

没有任何报错，表示文件上传成功。
